## Como abrir o template (.PBIT)

Os dados vêm de APIs públicas, e os usuários devem escolher a autenticação Anônima nas configurações da fonte de dados. Se o relatório exibir mensagens de privacidade, essas devem ser tratadas adequadamente. Além disso, as limitações de taxa/API podem afetar a atualização dos dados, e o acesso à API pode ser instável. 

Observação: os endpoints utilizados estão documentados em `docs/fontes.md`. 

Outras informações sobre o uso do template e funcionalidades estarão disponíveis nas seções seguintes.
